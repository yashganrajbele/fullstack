## 🧩 Features

### 🔐 Authentication & Security

* User registration and login
* JWT-based authentication
* Google OAuth2 login
* Forgot password / Reset password / Change password
* Email verification (OTP-based)

### 👥 Role-Based Access Control (RBAC)

* User and Admin roles
* Super Admin role with full control
* Secure role-based authorization

### 🔄 Admin Promotion Workflow

* Users can request admin access
* Super Admin can approve or reject requests
* Users can re-submit requests after rejection
* Status tracking for all requests

### ⚙️ Role Management

* Promote users to admin
* Demote admins back to users
* Full lifecycle management of roles

### 📡 Backend Functionality

* RESTful APIs for authentication and user management
* Secure endpoints with Spring Security
* Request tracking and status handling
